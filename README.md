# VitePress 学习笔记与个人博客

本项目使用 VitePress 搭建，旨在记录个人的学习笔记和技术思考。

> 星光不问赶路人，时光不负有心人。

## 快速开始

## 开发环境运行\*\*

```bash
npm run docs:dev
# 或者
yarn docs:dev
# 或者
pnpm docs:dev
```

## 构建静态站点\*\*

```bash
npm run docs:build
# 或者
yarn docs:build
# 或者
pnpm docs:build
```

## 本地预览构建后的站点\*\*

```bash
npm run docs:preview
# 或者
yarn docs:preview
# 或者
pnpm docs:preview
```
